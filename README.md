
# Studysync 📚

**Studysync** is a student-centric task organization mobile application designed to help learners manage their academic workload with ease. By combining subject-based organization with visual progress tracking, Studysync turns a daunting "to-do" list into a manageable, rewarding journey.

## 1. Project Overview

Managing multiple subjects and deadlines can be overwhelming. **Studysync** solves this by providing:
- **Centralized Task Hub:** A clean interface to view all upcoming tasks ordered by priority/date.
- **Subject-Based Organization:** Categorize tasks by subject (Math, Physics, etc.) to keep academic materials focused.
- **Visual Motivation:** Dynamic progress bars that update in real-time as tasks are completed, giving students a sense of accomplishment.
- **Modern Navigation:** An animated notch bottom bar for a fluid, professional user experience.

### Key Features
- **Interactive Checklists:** Easily mark tasks as complete with a line-through animation.
- **Progress Tracking:** View completion percentages for specific subjects and overall tasks.
